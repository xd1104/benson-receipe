設定 — GitHub 金鑰

貼上共用的 GitHub 金鑰後，這支手機就能新增／編輯食譜。

🔒 金鑰只存在這支手機的瀏覽器（localStorage），不會上傳、不會進程式碼；請只授權這一個食譜倉庫。
怎麼產生金鑰？（fine-grained PAT）
  1. 電腦或手機開 GitHub → 頭像 → Settings。
  2. 左下 Developer settings → Personal access tokens → Fine-grained tokens → Generate new token。
  3. Token name 隨意；Expiration 選一個期限。
  4. Repository access 選 Only select repositories → 只勾 benson-receipe。
  5. Permissions → Repository permissions → Contents 設為 Read and write。
  6. Generate → 複製那串 github_pat_... → 回到這裡貼上、儲存。
